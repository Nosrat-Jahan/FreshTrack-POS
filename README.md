# FreshTrack POS 🛒
A high-performance Grocery Inventory and Point of Sale (POS) system.

## ✨ Key Features
- **Smart Analytics:** Real-time tracking of revenue and net profit.
- **Inventory Engine:** Easy SKU entry with stock management.
- **Dark Orange Interface:** Modern, high-DPI supported UI.
- **Local Persistence:** Automated data syncing via JSON.

## 🛠️ Built With
- **Language:** Python 3.x
- **GUI:** Tkinter (Standard Library)
- **Data:** JSON Persistence Layer
